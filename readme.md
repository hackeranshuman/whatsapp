# 📱 WhatsApp Bulk Messenger

A professional Python desktop application for sending personalised WhatsApp messages to multiple clients — built with Tkinter, pandas, pywhatkit, and Twilio.

---

## ✨ Features

| Feature | Details |
|---|---|
| **CSV Import** | Upload `name, phone_number` CSV or use the downloadable template |
| **Manual Entry** | Add individual contacts via form fields |
| **Message Templates** | Use `{name}` placeholder — auto-replaced per recipient |
| **Live Preview** | See how your message looks before sending |
| **Two Send Backends** | pywhatkit (WhatsApp Web) or Twilio API |
| **Delay Control** | Set delay between messages (5–120 s) to avoid bans |
| **Send to All / Selected** | Flexible targeting |
| **Real-time Log** | Color-coded activity log per message |
| **Stop Button** | Gracefully cancel mid-batch sending |

## 📲 Sending Methods

### Option A — pywhatkit (WhatsApp Web) — Free, No API key
- Requires Google Chrome installed
- **Log in to WhatsApp Web** at https://web.whatsapp.com before clicking Send
- Opens a new Chrome tab per message, types and sends automatically
- Best for personal/occasional use

### Option B — Twilio API — Production-grade
1. Sign up at https://www.twilio.com
2. Enable the **WhatsApp Sandbox** (or get a dedicated WhatsApp number)
3. Go to **Settings → Twilio** tab in the app
4. Enter:
   - **Account SID** (from Twilio Console)
   - **Auth Token** (from Twilio Console)
   - **From Number** (e.g. `+14155238886` — your Twilio WhatsApp number)
5. Click **Test Twilio Connection** to verify
6. Start sending!

## 📂 CSV Format

```csv
name,phone_number
Alice Smith,+919876543210
Bob Jones,+14155238886
```

- Use the **"Download Template"** button in the app to get a ready-made example
- Phone numbers must include the country code (e.g., `+91` for India, `+1` for USA)

---

## 💬 Message Placeholders

| Placeholder | Replaced with |
|---|---|
| `{name}` | Recipient's name from CSV/list |

**Example:**
```
Hello {name}, your appointment is confirmed for tomorrow at 3 PM.
```

Becomes:
```
Hello Alice Smith, your appointment is confirmed for tomorrow at 3 PM.
```

## ⚠️ Notes & Best Practices

- **Delay**: Use at least 15 seconds between messages to avoid WhatsApp spam detection
- **pywhatkit**: Works best with a stable internet connection and Chrome
- **Twilio Sandbox**: Recipients must first send a join code to opt in
- **Bulk limits**: WhatsApp Business API has daily message limits — check Twilio docs
